obs-ssp
==============

Network A/V in OBS Studio with Simple Stream Protocol(SSP).

[简体中文](./README-zh.md)

## Features
- **SSP Source** : receive video and audio from ZCam cameras to OBS.

![obs-ssp](./images/obs-ssp.png)
